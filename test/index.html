Test for broken IndexedDB

This page tests the current browser's IndexexDB to see if it contains bugs.

Possible bugs you might see:

  • IndexedDB not supported at all
  • Samsung/HTC bug where IndexedDB doesn't have onupgradeneeded
  • Apple bug in UIWebView on iOS 8 where IDBKeyRange is null rather than undefined
  • Apple bug in desktop Safari 7.1+ and iOS 8+ Safari where IndexedDB doesn't allow opening two databases at once and is basically unusable.
